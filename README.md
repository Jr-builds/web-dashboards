# Web Dashboards

A collection of interactive business dashboards built from scratch.

## Dashboards included

### 1. Business Dashboard
Overview dashboard for business metrics — revenue, clients,
performance indicators and growth tracking.

### 2. CRM Dashboard
Customer relationship management dashboard — client pipeline,
deal tracking and contact management.

### 3. Coaching Dashboard
Dashboard for coaching businesses — client progress tracking,
session management and goal monitoring.

## Tech Stack
- HTML5 / CSS3 / JavaScript (vanilla)
- Responsive design
- Real-time interactive charts and metrics
- No frameworks — pure HTML/CSS/JS
